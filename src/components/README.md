## Ice Cream App

Bu proje ile  kullanıcıların dondurma menüsünü kullanıcıların menüyü görüntülemesini sağlamaları ve listeleyebilmeleri amaçlanmıştır..

## Kullanılan Teknolojiler

-React
-Axios 
-Bootstrap 
-json-server 
-Jest ve React Testing Library
